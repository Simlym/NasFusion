# 订阅与自动下载

订阅系统是 NasFusion 的核心功能，实现"一次配置，全自动追剧"。

## 创建订阅

进入 **订阅管理 → 新建订阅**：

| 字段 | 说明 |
|------|------|
| 媒体 | 搜索并选择电视剧（关联 TMDB） |
| 季度 | 选择要追的季（支持多季） |
| 质量偏好 | 1080p / 4K / 最佳画质 |
| 字幕偏好 | 中文字幕 / 国配 / 无要求 |
| 大小限制 | 单集最大文件大小 |
| 来源站点 | 从哪些 PT 站点匹配资源 |

## 自动匹配逻辑

每次资源同步后，系统自动：

1. 扫描新入库资源
2. 与活跃订阅进行匹配
3. 按质量优先级打分排序
4. 触发 qBittorrent 下载最优资源
5. 发送下载通知

## 质量优先级

```
4K HDR > 4K > 1080p REMUX > 1080p > 720p
```

同质量下优先选择：做种数更多 > 文件更小 > 上传时间更近

## 防重复下载

系统追踪每集的下载状态，已下载或正在下载的集数不会重复触发。

## 订阅状态

| 状态 | 说明 |
|------|------|
| 追更中 | 正在监听新资源 |
| 已完结 | 剧集完结，停止监听 |
| 暂停 | 手动暂停 |
| 已归档 | 全季下载完成 |

::: tip
可以在订阅详情页查看每集的匹配与下载状态，手动补充缺失集数。
:::
