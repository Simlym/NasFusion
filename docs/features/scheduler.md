# 任务调度

NasFusion 内置基于 APScheduler 的任务调度系统，管理所有后台自动化任务。

## 内置任务

| 任务 | 默认频率 | 说明 |
|------|----------|------|
| PT 资源同步 | 每 6 小时 | 同步所有已配置站点的资源列表 |
| 站点健康检查 | 每 1 小时 | 检测站点连接状态 |
| 订阅匹配 | 资源同步后触发 | 新资源与订阅进行匹配 |
| 下载任务监控 | 每 5 分钟 | 检查 qBittorrent 任务状态 |
| 媒体库同步 | 每天 2:00 | 与媒体服务器同步媒体信息 |
| 观看历史同步 | 每天 3:00 | 从媒体服务器拉取观看记录 |

## 任务管理界面

在 **任务调度** 页面可以：

- 查看所有任务的下次执行时间
- 手动立即触发任务
- 启用/禁用指定任务
- 修改任务执行频率
- 查看任务执行历史和日志

## 任务历史

每次任务执行都会记录：
- 开始/结束时间
- 执行结果（成功/失败）
- 处理数量（如"同步了 234 条资源"）
- 错误信息（失败时）

## 进度跟踪

长时间运行的任务（如批量同步）支持实时进度显示，可在界面上查看当前进度。
