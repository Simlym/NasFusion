# 资源同步与搜索

资源同步是 NasFusion 的数据基础。系统将 PT 站点的资源索引定期同步到本地数据库，使你能以毫秒级速度搜索和浏览资源，无需每次都实时访问 PT 站点。

## 同步原理

```
PT 站点（MTeam 等）
       │
       │  HTTP API（带 API Key）
       ▼
  同步任务（后台定时运行）
       │
       ├─ 拉取资源列表（分页）
       ├─ 解析资源信息（促销、季集等）
       ├─ 去重写入本地数据库（upsert）
       │
       ▼
  本地数据库（pt_resources 表）
       │
       ├──► 资源搜索（< 100ms）
       └──► 订阅自动匹配
```

## 自动同步

在 **设置 → 任务调度** 中配置自动同步：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 同步频率 | 每 6 小时 | 间隔时间，可调整 |
| 同步范围 | 全部站点 | 也可指定特定站点 |
| 每次同步页数 | 20 页 | 太多会增加 API 请求 |

::: tip 建议
- 同步频率不宜太高（避免 PT 站点封禁）
- 促销资源时效性高，可在促销期间临时加大频率
:::

进入 **资源管理 → 同步**，可以：

- **全量同步**：同步所有站点的最新资源
- **按站点同步**：只同步选定站点
- **按分类同步**：只同步特定分类（如只同步剧集）

### 同步进度

同步过程中可以实时查看进度：

```
正在同步：馒头
  页码：15 / 20
  本页资源：50 条
  已写入：732 条（其中新增 48 条，更新 684 条）
  耗时：23 秒
```

## 同步内容

每条资源记录包含以下信息：

| 字段 | 说明 |
|------|------|
| 标题 | 资源原始标题（含分辨率等信息） |
| 大小 | 文件大小（单个文件或整季） |
| 做种数 | 当前做种人数 |
| 下载数 | 历史下载次数 |
| 上传时间 | 资源发布时间 |
| 促销类型 | 免费 / 2X 上传 / 2X 免费 / 50% 下载等 |
| 促销截止 | 促销结束时间（若有） |
| 分辨率 | 1080p / 4K 等（从标题解析） |
| 编码 | H.264 / H.265 / REMUX 等 |
| 季 / 集 | 电视剧的季集信息（自动识别） |
| 分类 | 电影 / 剧集 / 动漫 等 |

## 去重机制

同一资源（相同 PT 站点内的相同 resource_id）只保留**最新状态**：

- 促销信息会随时间变化，每次同步会更新
- 做种数、下载数等统计信息也会更新
- 不会产生重复记录，数据库不会无限膨胀

系统自动识别并标注促销状态：

| 促销类型 | 标签颜色 | 说明 |
|----------|----------|------|
| 🆓 免费 | 绿色 | 下载不消耗流量 |
| 2️⃣ 2X 上传 | 蓝色 | 上传量计双倍 |
| ⭐ 2X 免费 | 金色 | 双重优惠（最划算） |
| 📉 50% 下载 | 橙色 | 下载量减半 |
| 📉 30% 下载 | 浅橙 | 下载量按 30% 计算 |

促销资源在订阅匹配中可以设置**优先选择免费资源**。

## 季集识别

系统从资源标题中自动解析季集信息，支持多种格式：

```
# 标准格式
The.Crown.S05E01.1080p.mkv  →  S05E01
权力的游戏 第一季 第01集    →  S01E01
Succession.S04.Complete     →  S04（整季）

# 中文格式
庆余年 第二季 第12集        →  S02E12
鱿鱼游戏 Season 2           →  S02

# 合集
Breaking.Bad.S01-S05        →  S01-S05（多季合集）
```

识别到季集信息后：
- 在资源列表中显示季集标签
- 订阅系统可以精确匹配到集

### 搜索方式

| 方式 | 说明 |
|------|------|
| **关键词搜索** | 匹配资源标题（支持中英文） |
| **分类筛选** | 电影 / 剧集 / 动漫 / 综艺 / 纪录片 |
| **促销筛选** | 只看免费 / 2X 等促销资源 |
| **来源站点** | 筛选特定 PT 站点的资源 |
| **分辨率** | 1080p / 4K / 其他 |

### 排序方式

- 上传时间（最新优先）
- 文件大小（从小/大）
- 做种数（最多优先）
