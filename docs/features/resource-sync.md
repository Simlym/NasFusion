# 资源同步与搜索

## 资源同步

NasFusion 定期将 PT 站点的资源列表同步到本地数据库，实现快速检索。

### 自动同步

在 **设置 → 任务调度** 中配置同步频率（默认每 6 小时同步一次）。

### 手动触发

在 **资源管理 → 同步** 页面手动触发，支持按站点、按分类选择同步范围。

### 同步内容

- 资源标题、大小、做种数、下载数
- 上传时间、促销信息（免费、2X 等）
- 分辨率、编码格式
- 季集信息（电视剧）

## 去重机制

同一资源在多次同步中只保留最新状态，基于资源 ID 去重，避免数据膨胀。

## 促销识别

自动识别站点促销标签：

| 促销类型 | 说明 |
|----------|------|
| 免费 | 下载不消耗流量 |
| 2X 上传 | 上传量计双倍 |
| 2X 免费 | 双重优惠 |
| 50% 下载 | 下载量减半 |

促销资源在列表中高亮显示，并可通过订阅系统优先匹配。

## 资源搜索

在 **资源搜索** 页面支持：

- 关键词搜索（标题）
- 按分类筛选（电影/剧集/动漫）
- 按促销类型筛选
- 按大小、做种数排序
- 结果直接触发下载
