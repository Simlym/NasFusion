# 通知系统

NasFusion 采用事件驱动的通知系统，关键操作完成后即时推送通知。

## 支持渠道

| 渠道 | 状态 |
|------|------|
| Telegram | ✅ 已支持 |
| 邮件（SMTP） | ✅ 已支持 |
| Webhook | 🚧 开发中 |
| 企业微信 | 🚧 计划中 |

## 触发事件

以下事件会触发通知：

| 事件 | 说明 |
|------|------|
| 下载完成 | 某个资源下载完毕 |
| 下载失败 | 任务出错 |
| 订阅匹配 | 新集数已找到并开始下载 |
| 站点异常 | PT 站点连接失败 |
| 磁盘空间不足 | 剩余空间低于阈值 |

## Telegram 配置

1. 与 [@BotFather](https://t.me/BotFather) 对话创建 Bot，获取 Token
2. 获取你的 Chat ID（可用 [@userinfobot](https://t.me/userinfobot)）
3. 在 **设置 → 通知 → Telegram** 中填写

## 通知模板

通知内容支持自定义，默认模板示例：

```
✅ 下载完成

《权力的游戏》S01E01
大小：8.5 GB
耗时：23 分钟
保存位置：/media/tv/权力的游戏/Season 1/
```

## 测试通知

配置完成后可点击 **发送测试通知** 验证是否正常工作。
