# 媒体库管理

## 媒体服务器集成

NasFusion 支持与主流媒体服务器对接：

| 媒体服务器 | 支持功能 |
|------------|----------|
| Jellyfin | 媒体库同步、观看历史、用户管理 |
| Emby | 媒体库同步、观看历史 |
| Plex | 媒体库同步、观看历史 |

### 配置方式

进入 **设置 → 媒体服务器**，填写：
- 服务器地址（如 `http://192.168.1.100:8096`）
- API Key（从媒体服务器设置页获取）

## 文件整理

下载完成后，NasFusion 可自动将文件整理到指定目录结构：

```
媒体库根目录/
├── 电影/
│   └── 复仇者联盟 (2012)/
│       └── 复仇者联盟 (2012).mkv
└── 电视剧/
    └── 权力的游戏/
        └── Season 1/
            ├── S01E01.mkv
            └── S01E02.mkv
```

### 整理规则

支持自定义整理规则：
- **硬链接**（推荐）：不占用额外空间，做种不受影响
- **软链接**：跨磁盘时使用
- **复制**：保留原文件

## 元数据刮削

整理完成后自动从 TMDB/豆瓣获取：
- 海报、背景图、剧照
- 简介、评分、演员表
- 中文译名

## 观看历史同步

从媒体服务器定期同步观看历史，用于：
- AI 推荐系统的数据来源
- 订阅完结检测（全季看完自动归档）
