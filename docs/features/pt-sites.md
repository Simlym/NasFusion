# PT 站点管理

PT 站点是 NasFusion 的数据来源。你可以配置多个 PT 站点，系统会定期同步各站点的资源到本地数据库。

## 支持的站点

| 站点类型 | 说明 | 状态 |
|----------|------|------|
| MTeam（馒头） | 内置完整适配器，支持所有功能 | ✅ 已支持 |
| NexusPHP 通用 | 基于 NexusPHP 框架的站点通用适配器 | 🚧 开发中 |
| 自定义适配器 | 继承 `BasePTSiteAdapter` 自行实现 | ✅ 支持扩展 |

## 添加站点

进入 **设置 → PT 站点 → 添加站点**：

<!-- 图：添加站点表单截图 -->
<!-- ![添加站点](/images/pt-sites/add-site-form.png) -->

::: tip 图片预留位置
*添加站点表单截图：展示站点名称、类型选择下拉框、API Key 输入框、代理地址输入框*
:::

| 字段 | 必填 | 说明 |
|------|------|------|
| 站点名称 | ✅ | 自定义名称，如"馒头" |
| 站点类型 | ✅ | 选择对应适配器 |
| API Key | ✅ | 从站点个人设置页获取（见下方） |
| 代理地址 | ❌ | 如 `http://127.0.0.1:7890`，不填则使用直连 |
| 备注 | ❌ | 可选描述 |

### 获取 MTeam API Key

1. 登录 MTeam 官网
2. 进入 **个人设置 → 实验室**
3. 找到 **API Key** 部分，点击生成
4. 复制生成的 API Key

## 测试连接

添加站点后，点击 **测试连接**，系统会：

1. 验证 API Key 有效性
2. 获取账户信息（上传量、下载量、魔力值、等级）
3. 测量站点响应时间

## 站点状态管理

### 站点列表

<!-- 图：站点列表页截图 -->
<!-- ![站点列表](/images/pt-sites/site-list.png) -->

::: tip 图片预留位置
*站点列表页截图：表格展示各站点名称、类型、状态（正常/异常）、最后检查时间、操作按钮*
:::

站点有以下状态：

| 状态 | 图标 | 说明 |
|------|------|------|
| 正常 | 🟢 | 最近健康检查通过 |
| 异常 | 🔴 | 连接失败或 API 报错 |
| 未知 | ⚪ | 尚未执行健康检查 |

### 健康检查

系统每 **1 小时**自动执行健康检查，也可手动点击 **立即检查**。

健康检查异常时，系统会通过通知渠道（Telegram/邮件）发出告警：

```
⚠️ PT 站点异常

站点：馒头
时间：2025-11-22 14:30
原因：API 请求超时（连续 3 次失败）

请检查 API Key 是否有效或网络是否正常。
```

## 分类同步

PT 站点有自己的分类体系（如 MTeam 的分类 ID 401、404 等），NasFusion 将其映射到标准分类。

点击 **同步分类** 按钮：

1. 拉取站点全部分类列表
2. 自动映射到标准分类（电影/电视剧/动漫/综艺/纪录片等）
3. 未识别的分类可手动配置映射关系

::: tip 图片预留位置

## 代理配置

每个站点可以独立配置代理，支持：

| 代理类型 | 格式 |
|----------|------|
| HTTP | `http://host:port` |
| HTTPS | `https://host:port` |
| SOCKS5 | `socks5://host:port` |
| 带认证 | `http://user:pass@host:port` |

::: tip 代理建议
- 对 MTeam 等国内可访问的站点，**不建议配置代理**（避免 IP 不匹配导致封号）
- 对访问受限的站点，**单独配置代理**而非全局代理
:::

你可以同时配置多个站点。资源同步时会按站点分别同步，搜索结果会显示来源站点：

<!-- 图：多站点资源列表截图 -->

::: tip 图片预留位置

## 扩展：添加自定义适配器

如需接入新的 PT 站点，继承 `BasePTSiteAdapter` 并实现以下方法：

```python
class MyPTSiteAdapter(BasePTSiteAdapter):
    async def test_connection(self) -> ConnectionResult:
        """测试连接，返回账户信息"""
        ...

    async def fetch_resources(self, page: int = 1) -> list[PTResource]:
        """获取资源列表"""
        ...

    async def get_resource_detail(self, resource_id: str) -> PTResourceDetail:
        """获取资源详情（磁力/种子）"""
        ...

    async def get_categories(self) -> list[PTCategory]:
        """获取站点分类列表"""
        ...
```
