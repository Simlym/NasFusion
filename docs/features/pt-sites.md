# PT 站点管理

NasFusion 支持管理多个 PT 站点，目前内置 MTeam 适配器，其他站点可通过自定义适配器接入。

## 添加站点

进入 **设置 → PT 站点 → 添加站点**，填写以下信息：

| 字段 | 说明 |
|------|------|
| 站点名称 | 自定义名称，如"馒头" |
| 站点类型 | 选择对应适配器（目前支持 MTeam） |
| API Key | 从站点个人设置页获取 |
| 代理地址 | 可选，如 `http://127.0.0.1:7890` |

## 连接测试

添加后点击 **测试连接**，系统会：

1. 验证 API Key 有效性
2. 获取用户信息（上传量、下载量、魔力值等）
3. 检测站点响应速度

## 健康检查

系统每小时自动对所有站点执行健康检查，状态异常时通过通知系统告警。

## 分类同步

点击 **同步分类** 可将站点分类同步到本地，并自动映射到标准分类（电影、剧集、动漫等）。

## 代理支持

每个站点可独立配置代理，支持 HTTP/HTTPS/SOCKS5 代理。

::: tip
建议为国内访问受限的站点单独配置代理，而非全局代理。
:::
