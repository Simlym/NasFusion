# 资源识别

资源识别系统负责将 PT 站点的原始资源标题解析为结构化的媒体信息，并与 TMDB/豆瓣的元数据进行关联。

## 为什么需要资源识别？

PT 站点的资源标题格式五花八门：

```
The.Crown.S05E01.2160p.UHD.BluRay.x265-GROUP
权力的游戏.Game.of.Thrones.S01E01.2011.1080P.BluRay.X264-中英双字
[VCB-Studio] Demon Slayer - Kimetsu no Yaiba [01][Ma10p_1080p][x265_flac]
《庆余年 第二季》第01集 1080p 中字
```

资源识别系统的工作是：

1. **解析标题** → 提取关键信息（标题、季集、分辨率等）
2. **匹配元数据** → 在 TMDB/豆瓣中找到对应条目
3. **建立关联** → 将 PT 资源与 TMDB 条目绑定

## 自动识别

资源同步后，系统会对新资源自动触发识别：

### 识别流程

```
原始标题："The.Crown.S05E01.2160p.UHD"
    │
    ▼
标题解析（清洗标题，去掉画质信息等）
    → 清洗标题："The Crown"
    → 季：5，集：1
    → 分辨率：4K
    → 编码：H.265
    │
    ▼
TMDB 搜索（使用清洗后的标题）
    → 搜索："The Crown"
    → 结果：TMDB ID 65494（英剧《王冠》）
    │
    ▼
置信度计算（标题相似度 + 季集匹配）
    → 置信度：95%（高置信度自动确认）
    │
    ▼
建立关联
    → pt_resources.tmdb_id = 65494
    → 获取中文名："王冠"
```

### 置信度阈值

| 置信度 | 处理方式 |
|--------|----------|
| ≥ 90% | 自动确认，无需人工 |
| 60% ~ 90% | 标记为「待确认」，人工审核 |
| < 60% | 标记为「识别失败」，需手动关联 |

## 识别管理页面

进入 **资源管理 → 识别**：

### 待确认列表

显示置信度在 60%-90% 之间、需要人工确认的资源：

<!-- 图：待确认列表截图 -->
<!-- ![待确认](/images/identification/pending-list.png) -->

::: tip 图片预留位置
*待确认列表截图：左侧显示 PT 资源信息，右侧显示 TMDB 候选结果（可能多个），用户选择正确的那个*
:::

对每条待确认记录，可以：
- **确认**：选择正确的 TMDB 匹配结果
- **搜索**：重新搜索 TMDB（修改关键词）
- **忽略**：跳过该资源（不建立关联）

### 识别失败列表

显示置信度过低、自动识别失败的资源，支持手动关联：

1. 点击资源，进入手动关联界面
2. 输入影视名称搜索 TMDB
3. 选择正确条目，点击确认

## 批量识别

对大量未识别资源，可以使用批量识别功能：

1. 进入 **资源管理 → 识别 → 批量识别**
2. 筛选要识别的范围（按站点、按分类、按时间）
3. 点击 **开始识别**
4. 系统后台处理，可在任务调度页查看进度

## 标题解析规则

系统支持识别以下格式的资源标题：

### 英文格式

```
The.Crown.S05E01.2160p.UHD.BluRay.x265-GROUP
Breaking Bad S04E11 1080p BluRay
Dune.Part.Two.2024.REMUX.2160p.HDR
```

解析结果：
- 剧名：去掉点号 → "The Crown"
- 季集：S05E01 → 第 5 季第 1 集
- 分辨率：2160p → 4K
- 编码：x265 → H.265
- 来源：BluRay

### 中文格式

```
《庆余年 第二季》第01集 1080p 中字
权力的游戏.第一季.2011.1080P.蓝光
[国语版] 流浪地球2 2023 4K REMUX
```

解析结果：
- 剧名：清洗后 → "庆余年"
- 季：第二季 → S02
- 集：第01集 → E01
- 字幕：中字 → 中文字幕

### 动漫格式

```
[SubsPlease] Jujutsu Kaisen - 47 (1080p) [ABC12345].mkv
[VCB-Studio] Demon Slayer [01][Ma10p_1080p][x265_flac]
【爱恋字幕组】[4K] 鬼灭之刃 刀匠村篇 第11话 [BDRip]
```

## 元数据来源

识别成功后，系统从以下来源获取元数据：

| 来源 | 优先级 | 内容 |
|------|--------|------|
| **TMDB** | 主要 | 国际通用信息、海报、评分 |
| **豆瓣** | 辅助 | 中文评分、中文演员名、国内上映信息 |

### TMDB 数据

- 官方标题（含中文）
- 简介（`zh-CN` 语言优先）
- 海报图（`zh-CN` > `en` > 其他）
- 背景图
- 演员表
- 导演/编剧
- 类型标签
- 首播日期

### 豆瓣数据（补充）

- 豆瓣评分
- 豆瓣中文演员名（更本地化）
- 制片国家/地区
- 国内上映日期
